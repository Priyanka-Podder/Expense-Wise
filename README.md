# Expense Wise 💰

A personal expense tracker web application for managing and visualizing daily expenses.

## Tech Stack
* **Frontend:** React
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

## Features
* Add, edit and delete expense entries with a clean interface 
* Track income / budget and expenses in real-time 
* View category-wise breakdown to better understand spending patterns  
* Interactive chart for visualizing better 

### Setup
1. **Configure:** Add credentials in `.env` file.
2. **Execute:** Run `npm install` and `npm run dev` in both client and server for local setup
